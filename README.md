# vue-low-code
## 低代码开发工具  个人开发使用  内包含了各种小工具
##  后端采用nodejs+express+mysql开发
##  默认用户名  admin   密码  password
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
